# Registrierungscodes - Band Planning Tool

## Gültiger Registrierungscode

Es gibt aktuell nur einen gültigen Registrierungscode:

"c2j5Dps!"

Dieser Code muss exakt so eingegeben werden (Groß-/Kleinschreibung beachten).

## Funktionsweise

1. Benutzer muss bei der Registrierung diesen Code eingeben
2. Der Code wird exakt geprüft
3. Nur bei Übereinstimmung kann die Registrierung fortgesetzt werden
4. Bei falschem Code erscheint die Fehlermeldung: "Ungültiger Registrierungscode"

## Sicherheitshinweis

> **Hinweis**: Der Code wird im LocalStorage gespeichert. In einer produktiven Umgebung sollte dies über ein sicheres Backend validiert werden.
